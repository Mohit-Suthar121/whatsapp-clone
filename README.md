## ✨ Overview

**ChatWave** is a full-stack **WhatsApp-inspired chat application** built using the **MERN stack**.  
It focuses on delivering a familiar, modern messaging experience with **real-time communication**, **OTP verification via phone number**, **email-based authentication**, and rich messaging features like **media sharing**, **reactions**, **reply**, **edit/delete**, and **seen/delivered message states**.

This project was built to replicate the core experience of WhatsApp while also demonstrating practical backend and real-time system design using technologies like **Socket.IO**, **JWT**, **Cloudinary**, **Twilio**, and **Nodemailer**.

---

## 🚀 Features

### 🔐 Authentication & Security
- Email-based authentication
- Phone number login / verification using **OTP**
- OTP delivery using **Twilio**
- Secure authentication using **JWT**
- Email workflows powered by **Nodemailer**

### 💬 Chat Features
- **1-to-1 private chat**
- **Real-time messaging** with **Socket.IO**
- **Typing indicator**
- **Online / Offline status**
- **Seen / Delivered message status**
- **Reply to messages**
- **Edit sent messages**
- **Delete messages**
- **Emoji support**
- **Message reactions**

### 📁 Media & Profile
- **Image upload**
- **Video upload**
- Media storage using **Cloudinary**
- **Profile update** support
